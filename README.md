# 李軍講堂 · 市場包裝專案

資深人才鑑別師「李軍講堂」在乾淨世界的會員訂閱節目《發現天賦 成就人生》的市場包裝資產，包含高轉化落地頁與市場策略文件。

## 內容

| 檔案 | 說明 |
|------|------|
| `李軍講堂_落地頁.html` | 高轉化落地頁（主檔，圖片用相對路徑引用，需與下方圖檔放在一起） |
| `李軍講堂_落地頁_可預覽單檔.html` | 同一頁面、圖片全內嵌的單檔版，任何地方都能開啟／分享 |
| `李軍講堂_市場策劃方案.docx` | 市場定位、受眾分層、產品價值階梯、轉化漏斗、文案資產等完整策劃 |
| `李軍講堂_引流與獨家會員節目策略建議.docx` | YouTube／專業渠道引流到乾淨世界做獨家會員節目的策略建議 |

## 如何檢視落地頁

直接用瀏覽器開啟 `李軍講堂_落地頁.html`（請與三張圖檔放在同一資料夾），或開啟 `李軍講堂_落地頁_可預覽單檔.html`（單檔即可）。頁面內嵌的課程預告片需連網才會顯示。

## 風格

頻道調性的淺灰＋深藍＋炭黑企業顧問風；案例區含以真實測評數據繪製的能力分析圖（SVG）。
